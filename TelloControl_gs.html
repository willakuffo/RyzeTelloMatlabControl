TelloControl

The TelloControl class is a MATLAB implementation of a simple software controller for the Ryze Tello drone. It provides an interface for sending commands to the drone, retrieving its state information, and receiving video stream.
To use this class, you need to create a TelloControl object and initialize it using the initTello() method. You can then use the various methods provided by the class to command the drone, retrieve its state, and receive video stream.
The TelloControl class has the following properties:
  • telloSDKPort: an integer representing the port number for Tello's SDK.
  • telloDroneStatePort: an integer representing the port number for Tello's drone state.
  • telloIP: a string representing Tello's IP address.
  • localhost: a string representing the local host IP address.
  • telloSDKClient: a udpport object representing the UDP client for Tello's SDK.
  • telloStateClient: a udpport object representing the UDP client for Tello's drone state.
  • telloCameraClient: a udpport object representing the UDP client for Tello's camera.
  • telloCameraPort: an integer representing the port number for Tello's camera.
  • telloState: a structure representing the state of Tello's drone
The TelloControl class provides the following methods:
  • Constructor: initializes the TelloControl object.
  • initTello: initializes the drone SDK.
  • cmdTello: sends a command to the drone.
  • getTelloState: retrieves the drone's state information.
  • motoron: turns on the drone's motors.
  • getCameraFeed: receives the drone's video stream. (not yet implemented)
Here's an example of how to use the TelloControl class:
% Create a TelloControl object
tc = TelloControl();
 
% Initialize the drone SDK
tc.initTello();
 
% Turn on the drone's motors
tc.motoron();
 
% Take off
tc.cmdTello('takeoff');
 
% Move forward
tc.cmdTello('forward 50');
 
% Turn right
tc.cmdTello('right 50');
 
% Land
tc.cmdTello('land');
 
Note that the getCameraFeed() method is not currently implemented in the code.
In conclusion, the TelloControl class provides a convenient way to control the Ryze Tello drone using MATLAB. It is recommended to use this code as a starting point for developing more complex software controllers for the drone.
